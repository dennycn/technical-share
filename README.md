typora支持的md格式与传统md格式的差别
1. md格式表格头前要有空行，typora没作要求。
2. 目录和参考链接也要保持显式换行。

**写作技巧**

1. 数字+空格 会变成有序符号且不需额外换行; 仅数字开头仍需换行。
